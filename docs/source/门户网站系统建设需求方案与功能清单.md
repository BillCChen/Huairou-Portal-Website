# 附件1：门户网站系统建设需求方案与交付清单

## 一、 项目概述与建设原则

### 1.1 项目概述

主门户网站作为北京怀柔科学城生命科学产业创新研究院的官方对外窗口，需要承担机构形象展示、信息发布、业务引导等核心功能。网站设计应体现研究院的专业性、权威性和创新性，通过现代化的视觉设计和用户体验设计，全面展示研究院在生命科学成果转化方面的成就。

### 1.2 建设原则

- 技术领先性：采用主流、稳定的技术架构，保证系统的先进性、安全性与可扩展性。
- 用户导向性：界面设计简洁直观，用户体验流畅，满足多角色用户的操作需求。
- 合规安全性：系统建设全过程符合国家网络安全与个人信息保护相关法律法规。

## 三、 技术、性能与安全要求

### 3.1 技术架构

- 采用前后端分离的微服务架构。
- 支持容器化（Docker）部署与编排（Kubernetes）。
- 数据库系统采用主流关系型数据库（如MySQL/PostgreSQL）。

### 3.2 性能指标

- 响应时间：
  - 普通页面加载及查询操作：≤ 2秒。
  - （测试环境：在标准配置的服务器上，于正常网络条件下测定）
- 并发能力：
  - 系统支持不少于 100 个用户同时在线进行核心业务操作。
  - （注：此为“核心业务操作”并发，非简单页面浏览。为精确、可验证的业务级并发指标）。

### 3.3 安全要求

- 系统部署防火墙、入侵检测等基础安全设施。
- 全部数据传输使用HTTPS加密。
- 用户密码需加密存储。
- 提供基于角色的权限访问控制（RBAC）。

## 六、 培训与运维服务方案

### 6.1 培训服务

- 内容：针对系统管理员、产业专班用户、专家用户，分别提供系统操作、后台管理等培训。
- 形式：提供1次现场集中培训 + 5次线上专题培训。
- 交付物：全套培训课件及培训录像。

### 6.2 运维服务（一年免费期）

- 服务范围：系统监控、故障排查、数据备份支持、系统漏洞修复。
- 服务级别协议（SLA）：
  - 一般故障：乙方在接到通知后2小时内响应，24小时内提出解决方案。
  - 重大故障（系统核心功能不可用）：乙方在接到通知后1小时内响应，48小时内提出解决方案。
  - （注：因甲方网络、硬件或第三方原因导致的故障，不在上述SLA承诺范围内）。
